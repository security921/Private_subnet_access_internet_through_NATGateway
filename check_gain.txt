# Integrating a PHP Web Application with Amazon DynamoDB Using AWS SDK


#Objective
The objective of this project was to integrate a PHP-based web application with Amazon DynamoDB and understand how a serverless NoSQL database can be used to store and retrieve application data. The project demonstrates how an EC2-hosted PHP website communicates with DynamoDB using the AWS SDK for PHP. 


#Architecture Overview
The architecture of this project includes the following AWS services:
* Amazon EC2
   * Hosts the PHP website.
   * Runs Ubuntu Linux, Nginx web server, and PHP-FPM.
   * Executes PHP code responsible for communicating with DynamoDB.
* Amazon DynamoDB
   * Used as the backend NoSQL database.
   * Stores website data in a scalable and highly available manner.
   * Provides low-latency access without managing database servers.
* AWS SDK for PHP
   * Provides libraries required for PHP applications to interact with AWS services.
   * Used to send API requests from the website to DynamoDB.
* Amazon S3
   * Used to store and download the project package during EC2 instance configuration.

<?php


require 'vendor/autoload.php';


use Aws\DynamoDb\DynamoDbClient;


$message = "";


$dynamodb = new DynamoDbClient([
    'version' => 'latest',
    'region' => 'eu-north-1'
]);


if ($_SERVER["REQUEST_METHOD"] == "POST") {


    $name = $_POST['name'];
    $email = $_POST['email'];
    $msg = $_POST['message'];


    try {


        $dynamodb->putItem([
            'TableName' => 'prakash_dyanamo_table',
            'Item' => [
                'id' => ['S' => uniqid()],
                'name' => ['S' => $name],
                'email' => ['S' => $email],
                'message' => ['S' => $msg],
                'created_at' => ['S' => date('Y-m-d H:i:s')]
            ]
        ]);


        echo "Submitted Successfully!";
        exit();


    } catch (Exception $e) {


        echo $e->getMessage();
        exit();


    }


} 


?>

# User Data Script
#!/bin/bash


apt-get update -y


apt-get install -y nginx php-fpm php-cli unzip wget composer php-curl php-xml php-mbstring


rm -f /var/www/html/index.nginx-debian.html


cd /tmp
wget https://prakash-dyanmodb.s3.eu-north-1.amazonaws.com/DynamoDb-porject.zip


if [ ! -f DynamoDb-porject.zip ]; then
    echo "Project download failed."
    exit 1
fi


unzip -o DynamoDb-porject.zip -d s3project


cp -r /tmp/s3project/DyanmoDB_web_site/* /var/www/html/


chown -R www-data:www-data /var/www/html
chmod -R 755 /var/www/html


# Only if vendor is NOT included in the ZIP
cd /var/www/html
if [ ! -d "vendor" ]; then
    composer require aws/aws-sdk-php --no-interaction
fi


PHP_VERSION=$(php -r "echo PHP_MAJOR_VERSION.'.'.PHP_MINOR_VERSION;")


cat >/etc/nginx/sites-available/default <<EOF
server {
    listen 80;
    server_name _;


    root /var/www/html;
    index index.php index.html;


    location / {
        try_files \$uri \$uri/ /index.php?\$query_string;
    }


    location ~ \.php\$ {
        include snippets/fastcgi-php.conf;
        fastcgi_pass unix:/run/php/php${PHP_VERSION}-fpm.sock;
    }
}
EOF


if nginx -t; then
    systemctl restart php${PHP_VERSION}-fpm
    systemctl restart nginx
    systemctl enable nginx
    systemctl enable php${PHP_VERSION}-fpm
else
    echo "Nginx configuration failed."
    exit 1
fi
